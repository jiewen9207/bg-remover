# Image Background Remover MVP 需求文档

## 1. 项目概述

**项目名称**：BG Remover（待定）

**核心定位**：一个简单、高效的在线图片背景去除工具，用户无需注册即可使用。

**核心价值**：让用户用最快的速度，得到一张透明背景的图片。

**目标用户**：
- 电商卖家（快速制作白底图）
- 设计师（快速获取素材）
- 普通用户（证件照换底色等）

---

## 2. 功能需求

### 2.1 核心功能（Must Have）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传，支持 JPG/PNG/WebP，单张≤10MB | P0 |
| AI 去背景 | 调用 remove.bg API，一键去除背景 | P0 |
| 预览对比 | 去背景前后对比展示 | P0 |
| 图片下载 | 下载透明背景 PNG 图片 | P0 |
| 进度提示 | 上传中、处理中、完成 各阶段状态提示 | P0 |

### 2.2 扩展功能（Should Have）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 背景颜色切换 | 去除背景后，可选换成白底/其他颜色 | P1 |
| 响应式适配 | 移动端正常显示和操作 | P1 |
| 历史记录 | 浏览器本地存储最近10条处理记录 | P2 |

### 2.3 不做（Won't Have）

- 用户登录/注册
- 批量处理
- API 开放
- 付费订阅
- 微信/支付宝支付

---

## 3. 技术方案

### 3.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML + Vanilla JS | 轻量、无构建依赖 |
| 后端 | Cloudflare Workers | 接收图片、调用 API、返回结果 |
| AI 能力 | remove.bg API | 成熟的背景去除服务 |
| 部署 | Cloudflare Pages + Workers | 全球 CDN 加速 |

### 3.2 系统架构

```
用户浏览器
    │
    ▼
Cloudflare Pages（前端页面）
    │
    ▼
Cloudflare Workers（/api/removebg）
    │
    ├── 接收图片（内存流）
    ├── 调用 remove.bg API
    └── 返回结果给前端
```

### 3.3 API 接口设计

**POST /api/removebg**

请求：表单上传图片文件 (`file` 字段)
响应：
- 成功：返回处理后的图片二进制流 (`Content-Type: image/png`)
- 失败：返回 JSON `{ "error": "错误描述" }`

### 3.4 限制与约束

- 单次图片大小：≤ 10MB
- 处理超时：30 秒
- Workers 内存：≤ 128MB
- 每日免费调用：remove.bg 免费额度 50 张/月

## 4. UI/UX 设计

### 4.1 页面结构

**单页面设计**，所有操作在一个页面完成：

```
┌─────────────────────────────┐
│          LOGO               │
├─────────────────────────────┤
│                             │
│   [上传区域 / 预览区域]      │
│                             │
├─────────────────────────────┤
│      [下载按钮]             │
├─────────────────────────────┤
│          版权信息           │
└─────────────────────────────┘
```

### 4.2 交互流程

1. 用户进入页面 → 显示上传区域
2. 用户上传图片 → 显示预览（原图）
3. 自动触发处理 → 显示 loading
4. 处理完成 → 显示去背景后的预览
5. 用户点击下载 → 浏览器下载图片

### 4.3 设计风格

- 风格：极简、干净
- 主色调：白底 + 蓝/紫色点缀
- 字体：系统默认字体
- 无需登录、无广告、无弹窗

---

## 5. 非功能需求

### 5.1 性能

- 首屏加载：≤ 2 秒
- 图片处理：≤ 30 秒
- 全球可访问，CDN 加速

### 5.2 安全

- 不在服务端存储任何图片
- 不收集用户任何个人信息
- 防止大文件攻击（前端限制 10MB）

### 5.3 兼容性

- 支持 Chrome、Firefox、Safari、Edge 最新两个版本
- 支持 iOS Safari、Android Chrome
